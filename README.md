# OS_System
(Java + MySQL)
## Sobre

Este projeto é um sistema de ordem de serviço desenvolvido em Java, com integração ao banco de dados MySQL.
Até o momento, foi implementada a conexão com o banco de dados, servindo como base para as próximas funcionalidades do sistema.
